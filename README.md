# R1 Web Serial Debugger

面向 R1 遥控器、底盘和定位板的浏览器串口观测工具。队友只需打开 HTTPS 网站，使用桌面版 Chrome 或 Edge 手动选择本机串口；不需要安装 Python 上位机。

## 功能

- 通信诊断：同时读取遥控器 RDBG 与底盘 CDBG，完整迁移本地 Python 工具的“遥控器链路 / 底盘 NRF / 定位输入”三组诊断、综合结论和阈值状态色。
- 指标悬停：每个诊断指标都能通过鼠标、键盘或触屏展开“含义 / 正常范围 / 异常判断 / 优先排查 / 源码字段”。
- 定位地图：读取 `$R1M`、CSV v2/v3 和 legacy CSV，直接使用冻结 Python 上位机的 2430×2420 场地图和 R1 机器人贴图，绘制 Final/Calib/LiDAR 三轨迹及 DT35 实测、期望、残差和模型裁剪。
- 地图交互：七类图层开关、跟随机器人、鼠标锚点缩放、拖拽平移、世界坐标悬停、机器人/场地模型/DT35 详情与地图截图。
- 数据示波器：在共享真实时间轴上单选或多选 Remote、Chassis、Locator 任意数值字段；支持十字线读数、图例显隐、10 秒到 30 分钟窗口、自动/固定 Y、框选和滚轮缩放、Shift 平移、双击复位、暂停显示和快速组合。
- 回放：打开 `.log`、`.txt`、`.csv` 或本站导出的 `.zip`，支持播放、暂停、单步、进度拖动和 0.1–5 倍速。
- 本地记录：采集分片写入浏览器 OPFS，异常刷新后可重新下载；停止时生成 Python 兼容文件名的 ZIP。
- 离线报告：通信页可导出 Markdown/HTML 诊断报告；定位页可导出当前状态 JSON、可见日志和地图 PNG。

本站严格接收：代码不创建串口 writer、不发送命令，也不调用 `setSignals()`。USB 转串口驱动在打开端口时是否改变线路状态仍取决于具体硬件和驱动，首次使用必须观察板卡是否复位。

## 使用

1. 用 Windows Chrome/Edge 打开正式 `pages.dev` HTTPS 地址。
2. 在“通信诊断”或“定位地图”点击“选择串口”，在浏览器弹窗中手动选择对应设备，再点击“连接”。端口号不会写死，网页也不会自动交换选错的设备。
3. 连接后页面会分别显示“尚无数据 / 有字节等待有效帧 / 数据正常 / 数据过期 / 疑似选错端口”，并展示 USB VID/PID（浏览器能够提供时）。
4. 切换到“数据示波器”。工作区常驻，切换页面不会断开已连接串口；左侧会自动出现当前协议的全部数值字段。可逐项勾选，也可使用“链路质量 / 双端信道 / 定位 XY / 姿态角 / DT35 / 驱动反馈”快速组合。
5. 地图中滚轮缩放、拖拽平移，悬停机器人、DT35 射线或场地模型可查看详情；右上角控制图层与跟随。
6. 没有硬件时可点击各页的“演示数据 / 演示轨迹 / 演示波形”检查完整 UI 和交互。
7. 需要完整记录时先点击“开始本地录制”，结束时点击“停止并下载”。下载路径由浏览器设置决定；此电脑若浏览器沿用系统下载目录，通常落在 `D:\网站下载`，网页本身不能强制指定目录。

Web Serial 必须运行在安全上下文中。Firefox、Safari、普通 HTTP 和直接双击 `file://` 不支持实时串口，但仍可在兼容环境使用文件回放。

## 数据与版本边界

- 串口数据只在当前浏览器内处理，不上传、不接遥测。
- OPFS 是站点私有临时/恢复存储；用户可见文件仅通过普通浏览器下载产生。
- 通信/定位 UI 最多保留 2000 行，地图每类最多 5000 个轨迹点；完整 OPFS 录制不随显示降采样。
- 示波器最多保留 30 分钟或每序列 120000 点，绘图前以 min/max bucket 保留峰值并限制到每序列 10000 点，UI 更新最高 30 FPS。
- Remote 行为冻结到 `cd0b796e392493c7c1acc46b5059f37c46bd0a66`。
- Locater 行为冻结到 `e6850444fdc3872951a64f8af035b1cffc099a94`；网页显示协议提供的原始 Final 结果，不引入冻结提交之后未提交的融合行为。

## 发布

站点为纯静态构建，不需要后端：

```powershell
npm run build
npx wrangler login
npx wrangler pages deploy dist --project-name r1-web-serial-debugger
```

Cloudflare OAuth 和浏览器串口授权必须由用户本人确认。仓库与 CI 不保存 Cloudflare/R2 密钥。
